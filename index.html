🐙 إعداد المزامنة عبر GitHub

بيانات اللوحة تُحفظ في GitHub وتنعكس فوراً على جميع من يفتح الرابط.

خطوات الإعداد (مرة واحدة فقط):
1. أنشئ حساباً في github.com
2. أنشئ Repository جديد — اختر Public
3. أنشئ ملف data.json في الـ Repository وضع فيه: {"_v":"hdiyt_v2"}
4. فعّل GitHub Pages: Settings → Pages → main branch
5. أنشئ Personal Access Token: Settings → Developer settings → Tokens (classic) → New → فعّل repo
